# 🍄 Mario Calculator

A mobile-first React calculator app with 7 powerful calculators in a fun Mario-themed interface.

## 🎮 Calculators

| Icon | Name | Description |
|------|------|-------------|
| 🧮 | **General** | Basic math with keypad, %, !, parentheses |
| 🏦 | **EMI** | Loan EMI with principal/interest breakdown |
| 🎂 | **Age** | Age finder with total days, weeks, next birthday |
| ⚖️ | **BMI** | Body Mass Index with visual category scale |
| 🔥 | **Calorie** | Daily calorie needs (BMR/TDEE) with activity levels |
| 🧾 | **VAT** | Add or remove VAT with preset rates |
| 📈 | **SMV** | Stock market value with profit/loss % |

## ✨ Features

- 📱 **Mobile-first** design optimized for phones
- 💾 **Save to localStorage** — history persists across sessions
- 📤 **Share via WhatsApp** — formatted result messages
- 🕐 **History panel** per calculator with clear option
- 🍄 **Mario pixel theme** with animations

## 🚀 Getting Started

```bash
npm install
npm run dev
```

## 🏗️ Build

```bash
npm run build
```

## 📦 Tech Stack

- **React 18** + **Vite**
- Vanilla CSS-in-JS (no extra libraries)
- localStorage for persistence
- Google Fonts: Press Start 2P + Nunito
